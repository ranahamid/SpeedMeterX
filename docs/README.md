# Documentation Assets

Place screenshots here for the README:

- `screenshot.png` - Main app screenshot for README header
- `running.png` - Speed test in progress
- `complete.png` - Test results view
- `paused.png` - Paused state

## How to Capture Screenshots

1. Run the application
2. Use browser dev tools to set viewport to 1280x800 for consistency
3. Capture screenshots using your OS screenshot tool or browser extension
4. Save as PNG files in this folder
